Practical Day 2

1. Write a program to create functions that will do the following 1. Add 2 numbers 2. Subtract 2 numbers 3. Divide 2 numbers 4. Multiply 2 numbers
Then update the program to get dynamic no of arguments(there can be 2,3,4 etc no of variables passed in when invoking the functions) The function should accept at least 2 variables and throw error if less then 2 are passed. The function should also validate the type of argument passed in.

Call function for static number of arguments: arithmeticFunctionWithStaticArgument(Operation, argument1, argument2)     in console
Operation can be 'Add', 'Subtract', 'Multiply', 'Divide'


Call function for dynamic number of arguments: arithmeticFunctionWithDynamicArgument(Operation, argument1, argument2,..., argumentN)      in console
Operation can be 'Add', 'Subtract', 'Multiply', 'Divide'
